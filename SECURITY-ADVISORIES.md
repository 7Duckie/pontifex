# Security advisory log

This file tracks every CVE / GHSA / security advisory that has been
evaluated against Pontifex, with disposition and rationale.

Format: one section per advisory, newest at top. Each entry includes:

- Identifier (CVE-XXXX-XXXXX or GHSA-xxxx-xxxx-xxxx)
- Affected package and version range
- Severity (CVSS) and exploit probability (EPSS, if available)
- Disposition: Patch / Defer / Not applicable / Upstream pending
- Rationale
- Re-review date (if Defer or Upstream pending)
